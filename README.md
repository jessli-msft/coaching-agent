# coaching-agent
Repo is a constantly revolving POC for the coaching agent through the phases.

Executive Summary:
Agent Coach is an always-on, in-the-moment coaching capability embedded within Dynamics 365 Customer Service that transforms quality insights and governance guidance into actionable, contextual coaching for customer service representatives. Built on top of the Quality Evaluation Agent (QEA) and Governance Agent, Agent Coach helps representatives improve performance case-by-case, while giving supervisors scalable tools to configure coaching strategy, ensure compliance, and accelerate skill development. The solution closes the loop between evaluation, coaching, and measurable improvement without disrupting the representative workflow.
